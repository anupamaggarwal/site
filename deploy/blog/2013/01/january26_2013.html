A cup of hyde jinja and bootstrap

Posted on 26 Jan 2013
This is my entry

My experiments with getting started with hyde My first post is going to be a genesis story[^1]. I wanted to use github to host a technical blog and saw that github pages officially supports jekyll. But let’s be honest I’m not going to work on a Ruby project. Enter Hyde. See what they did there? Hyde is a static website generator written in Python that supports Jinja2 templates. Luckily, I have some familiarity with Jinja from working on the statsmodels documentation that’s generated by Sphinx.

This isn’t a how to get started or a migrating from Wordpress or Django post. There are several of those already. I’m going to outline some changes I’ve made to existing Hyde tools to arrive at my workflow for blogging and publishing this site to github pages.
